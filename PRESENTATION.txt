# MemeGen - AI-Powered Meme Generator

## Project Overview

MemeGen is a modern, AI-powered web application that allows users to create memes instantly using artificial intelligence. Built with React, Vite, and Google's Gemini API, it combines the fun of meme culture with the power of AI to generate witty captions automatically.

---

## What This Project Does

1. **Meme Template Gallery**
   - Fetches popular meme templates from the Imgflip API
   - Displays a searchable gallery of meme templates
   - Users can browse and select any template to edit

2. **AI-Powered Caption Generation**
   - Uses Google's Gemini 3 Flash AI model
   - Automatically generates funny, relevant captions for selected memes
   - Provides both top and bottom text for classic meme format

3. **Meme Editor**
   - Customize text with font size and color options
   - Toggle text outline/shadow for readability
   - Real-time preview of the meme
   - Download the final meme as a PNG image

## How to Run

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up your API key:
   - Create a `.env` file in the project root
   - Add your Google Gemini API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to the URL shown (usually http://localhost:5173)

## API Integration

### Imgflip API
- Used to fetch the library of meme templates
- Endpoint: `https://api.imgflip.com/get_memes`
- Returns array of meme objects with ID, name, URL, dimensions

### Google Gemini API
- Used for AI-powered caption generation
- Model: `gemini-3-flash-preview`
- Generates JSON response with `topText` and `bottomText`
